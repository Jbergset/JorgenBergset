Bachelor prosjekt: Differ Bot

Dette var oppgaven:

Som oppgave på bachelor prosjektet så hadde vi som oppdrag å utvikle en chatbot og ett admin panel for selskapet Ed-Tech Foundry. Chatbotten skal kunne svare på generelle spørsmål studenter trenger raskt svar på, slik at forelesere skal kunne slippe å måtte svare på de mest generelle spørsmålene. Hvis det var ett spørsmål ikke chatboten kan svare på så vil det spørsmålet bli sendt til admin panelet der foreleser kan svare på spørsmålet slik at chatboten vet hva den skal svare neste gang en student stiller ett lignende spørsmål.

Slik ble den løst:

Min oppgave under bachelor-prosjektet var å utvikle admin panelet og data mine de meste spurte spørsmålene fra student til foreleser. For admin panelet så utviklet vi det i Javascript med rammeverket React. Og med data mining så brukte jeg R og Excel for å hente ut de meste spurte spørsmålene som vi brukte til å trene chatboten til å kunne svare på.
